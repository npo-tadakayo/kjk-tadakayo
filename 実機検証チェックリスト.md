# タダカヨCRM 実機検証チェックリスト

> 対象: https://kjk-tadakayo-admin.web.app
> 前提: @tadakayo.jp の登録済みアカウントでログイン（未登録は拒否される＝正常）
> 使い方: 上から順に確認し、問題があれば「気づき」欄にメモ。Claudeが次セッションで修正します。

> [!INFO]
> 多くの機能は前回までコードは完成しているが @tadakayo ログインが要るため Claude では未検証です。ここを次田さんが一度通すことで「本当に動くか」を確定します。

---

## 0. ログイン・権限（rule05 認証系は必ず実機で）

- [ ] @tadakayo.jp（管理者: 次田/佐藤）でログイン → 案件一覧が出る
- [ ] 未登録の @tadakayo.jp アカウントでログイン → 「アクセス権がありません」で拒否される
- [ ] staff アカウントでログイン → 「設定」「ユーザー管理」が拒否される（adminOnlyのガードがOK）
- [ ] ログアウト → 再ログインの3ステップが通る
- [ ] （任意）ログイン時のCOOP警告（`window.closed`）が Safari で実害ないか

## 1. 案件管理

- [ ] 案件一覧が表示（20件前後）／受信日・最終更新が **西暦 YYYY/MM/DD** 表示
- [ ] 検索・流入元/ステータスの絞り込みが効く
- [ ] 「CSV出力」→ 絞り込み結果がExcelで開ける（文字化けなし）
- [ ] 案件詳細 → タイムライン・書類チェック・申請情報タブが出る
- [ ] 案件詳細の口座情報（事業所側=補助金受取口座）が保存できる

## 2. カンバン

- [ ] 13ステータス列に案件が並ぶ
- [ ] カードをドラッグ&ドロップ → ステータスが変わり、詳細タイムラインに記録される
- [ ] 上部3チップ（未割当／未申請／停滞7日）の集計が妥当

## 3. ダッシュボード

- [ ] サマリー（総数/受注/申請/完了/失注/想定補助額/CR台数）が妥当
- [ ] パイプラインファネル・ステータス別/流入元別バーが出る

## 4. 伴走支援・写真

- [ ] 案件詳細「伴走支援」タブで実施日＋メモ＋写真を保存できる
- [ ] スマホで「カメラで撮影」→ カメラが起動する（`capture=environment`）
- [ ] 写真サムネ表示・クリックで原寸／タイムラインに訪問記録が出る

## 5. 報告書PDF

- [ ] 案件詳細「報告書PDF」→ 別タブで内容表示 → 印刷/PDF保存できる

## 6. 供給管理（在庫・発注・出荷）

- [ ] 在庫・商品: 3SKUの在庫表示＋手動±調整が記録される
- [ ] 新規発注 → 採番 `PO-2026-NNNN` → **発注書PDF**が出る
- [ ] 発注書PDFの角印が **実印影画像**で出る（文字印でなく）※初回発注時に要確認
- [ ] 入荷登録 → 在庫が増える
- [ ] 新規出荷 → 採番 `SH-2026-NNNN` → 在庫が減る → **送付状PDF**が出る

## 7. 受注→発送→請求→入金（業務フロー）

- [ ] 受注タブ（認定事業所）で partnerOrder が見える
- [ ] 受注「出荷へ」→ 在庫が減り、dropship 出荷が生成（卸価格スナップショット）
- [ ] 出荷タブ「請求済にする」「入金記録（金額・日付）」が動く
- [ ] サマリー（未請求/請求済・未入金/入金済）の税込金額が合う

## 8. 請求書・宛名PDF

- [ ] 請求書PDF（直接支援=定価／直送=卸価格）が正しい単価で出る
- [ ] インボイス登録番号が出る（**設定で T+13桁を登録後**）
- [ ] **お振込先が実際の口座で出る（※本セッションのデプロイ反映後・下の「9.設定」で登録）**
- [ ] レターパック宛名（赤600円/青370円）＋差出人選択 → 実物準拠で出る
- [ ] 認定事業所「宛名」ボタン → 登録住所からレターパック宛名が出る

## 9. 設定（未入力項目の登録）★次田さん入力待ち

- [ ] Chat Webhook URL → 「Chatにテスト通知」で着弾する
- [ ] メール送信元（kjk-staff@tadakayo.jp）
- [ ] 差出人リスト（レターパック差出人）
- [ ] 発注書: 発行元名・住所・代表者・発注者氏名・印影文字（印影画像は登録済）
- [ ] 適格請求書: 発行事業者名・**登録番号（T+13桁）**
- [ ] **請求書の振込先口座: 金融機関名・支店・種別・口座番号・名義（★今回追加）**

## 10. AIアシスタント・メール

- [ ] 案件詳細「AIアシスタント」→ 返信下書き／要約・分類／伴走報告文／自由質問が生成される
- [ ] AI下書き → 「送信欄へ転記」→ メール送信（最初は自分宛てが安全）→ タイムラインに送信記録

## 11. モバイルUI（スマホで）

- [ ] ヘッダーのハンバーガー → ドロワー開閉・ナビ遷移
- [ ] 伴走タブのカメラ撮影がスマホで動く

## 12. ユーザー管理・ヘルプ

- [ ] ユーザー管理で氏名/ロールを編集できる（藤田/斎藤/石田/蜂須賀の漢字氏名は仮 → 要修正）
- [ ] 自分の削除/停止/降格に確認が出る（自己ロックアウト防止）
- [ ] サイドバーの「マニュアル」「エンジニアノート」が表示／エンジニアノートのMermaid 4図が描画される
- [ ] 各画面タブにタダカヨロゴのfavicon

## 13. B2Bポータル（認定事業所向け・要・許可リスト登録アカウント）

- [ ] 供給管理→パートナー管理 で認定事業所のログインメール＋名称を追加
- [ ] そのメール本人が /partner.html にGoogleログイン → 発注フォーム＋自分の発注履歴が見える
