## デザイン指示

- **全体トーン**: 明るく・信頼感・NPO らしい温かみ
- **メインコピー**は画面の 40〜50% を占める大きさで（視認性最優先）
- **「¥0」** の部分は赤（`#E33535`）で強調
- キャラクターは **吹き出しなし**でシンプルに配置（吹き出しは Web 側で表示済みのため）
- 背景は **白〜淡ピンクのグラデーション**（Web サイトのヒーローセクションと統一）
- SNS での縮小表示（約 300×158px）でもテキストが読めるサイズ感で

## 納品後の設置手順

1. `images/ogp.png` に保存
2. `index.html` の `<head>` に以下を追加：
   ```html
   <meta property="og:image" content="https://kjk.tadakayo.jp/images/ogp.png">
   <meta property="og:image:width" content="1200">
   <meta property="og:image:height" content="630">
   <meta name="twitter:image" content="https://kjk.tadakayo.jp/images/ogp.png">
   ```
3. Firebase Hosting にデプロイ
4. [OGP 確認ツール](https://rakko.tools/tools/9/) で表示を確認
