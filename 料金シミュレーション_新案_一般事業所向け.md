## ■ 補足：複数サービス提供事業所の合算例

同一事業所で複数サービスを提供している場合、各種別の上限を合算可能。

| 事業所の保有サービス | 合算上限 | 最大台数 |
|---|---|---|
| 訪問 + 施設（居住） | ¥64,000 + ¥55,000 = **¥119,000** | 3+2 = 5台 |
| 訪問 + その他 | ¥64,000 + ¥42,000 = **¥106,000** | 3+1 = 4台 |
| 施設 + その他 | ¥55,000 + ¥42,000 = **¥97,000** | 2+1 = 3台 |
| 居宅介護支援 + 介護予防支援（事業所番号別）| ¥64,000 × 2 = **¥128,000** | 3+3 = 6台 |
| 福祉用具貸与（17）＋特定福祉用具販売（41）同一事業所番号 | ¥64,000 + ¥20,000 = **¥84,000** | 2台 |

> ※ 合算の場合は各種別ごとに申請書類が必要となる場合があります。詳細は介護情報基盤ポータルにてご確認ください。
