## 3. 発注ファイル仕様

認定事業所からの発注は **CSVファイル** または **JSONファイル** のどちらかでご送付ください。  
メール送付先: `yoshinao-tsukuda@tadakayo.jp`（タダカヨ次田）  
件名フォーマット: `【発注】{認定事業所名} {発注日YYYYMMDD}`

---

### 3-1. CSV形式（推奨・Excelで編集可能）

**ファイル名**: `発注_{認定事業所ID}_{YYYYMMDD}.csv`  
**文字コード**: UTF-8（BOM付き推奨）  
**1行目**: ヘッダー行（固定）  
**2行目以降**: 注文明細（1行 = 1品番）

#### ヘッダー定義

| 列名 | 型 | 必須 | 説明 | 例 |
|---|---|---|---|---|
| `partner_id` | 文字列 | ✅ | 認定事業所ID（タダカヨが発行） | `PARTNER-001` |
| `partner_order_no` | 文字列 | ✅ | 認定事業所側の管理番号 | `JO-2026-042` |
| `order_date` | 日付 | ✅ | 発注日 YYYY-MM-DD | `2026-05-27` |
| `contact_name` | 文字列 | ✅ | 担当者名 | `山田 花子` |
| `contact_phone` | 文字列 | ✅ | 担当者電話番号 | `03-1234-5678` |
| `contact_email` | 文字列 | ✅ | 担当者メールアドレス | `yamada@example.com` |
| `sku` | 文字列 | ✅ | 商品SKU（`cir415a-01` または `cir315a-01`） | `cir415a-01` |
| `qty` | 整数 | ✅ | 数量 | `3` |
| `desired_delivery_date` | 日付 | ✅ | 希望納期 YYYY-MM-DD | `2026-06-10` |
| `shipping_type` | 文字列 | ✅ | 送付先種別（下表参照） | `direct` |
| `shipping_postal_code` | 文字列 | 条件付き | 送付先郵便番号（`direct`/`other`の場合必須） | `130-0001` |
| `shipping_address` | 文字列 | 条件付き | 送付先住所（`direct`/`other`の場合必須） | `東京都墨田区吾妻橋1-1-1` |
| `shipping_company` | 文字列 | 条件付き | 送付先法人名 | `社会福祉法人〇〇会` |
| `shipping_office_name` | 文字列 | 条件付き | 送付先事業所名 | `〇〇デイサービス` |
| `shipping_contact_name` | 文字列 | 条件付き | 送付先担当者名 | `鈴木 一郎` |
| `shipping_contact_phone` | 文字列 | 条件付き | 送付先電話番号 | `03-9876-5432` |
| `is_subsidy_applied` | 真偽値 | ✅ | 助成金申請対象か（`true`/`false`） | `true` |
| `subsidy_category` | 文字列 | 条件付き | 補助金区分（`houmon`/`kyojyu`/`other`）※is_subsidy_applied=trueの場合必須 | `houmon` |
| `notes` | 文字列 | - | 補足メモ | `急ぎ対応希望` |

#### `shipping_type` の値

| 値 | 意味 |
|---|---|
| `partner` | 認定事業所の本社・事業所宛（事前登録住所） |
| `direct` | 介護事業所へ直送（住所入力必須） |
| `other` | その他指定先（住所入力必須） |

#### `subsidy_category` の値

| 値 | 区分 | 補助上限 |
|---|---|---|
| `houmon` | 訪問・通所・短期滞在系（居宅介護支援含む） | ¥64,000 |
| `kyojyu` | 居住・入所系（グループホーム・特養等） | ¥55,000 |
| `other` | その他 | ¥42,000 |

#### CSVサンプル（2明細）

```csv
partner_id,partner_order_no,order_date,contact_name,contact_phone,contact_email,sku,qty,desired_delivery_date,shipping_type,shipping_postal_code,shipping_address,shipping_company,shipping_office_name,shipping_contact_name,shipping_contact_phone,is_subsidy_applied,subsidy_category,notes
PARTNER-001,JO-2026-042,2026-05-27,山田 花子,03-1234-5678,yamada@example.com,cir415a-01,3,2026-06-10,direct,130-0001,東京都墨田区吾妻橋1-1-1,社会福祉法人〇〇会,〇〇デイサービス,鈴木 一郎,03-9876-5432,true,houmon,
```

## 5. よくある質問

**Q. 複数の介護事業所へ同時に発注できますか？**  
A. 可能です。発注先ごとに1行（CSV）または `items`配列（JSON）を分けて、`shipping` に各事業所の住所を記載してください。ただし、`partner_order_no` は1送付先 = 1番号で管理することを推奨します。

**Q. 最小発注数量は？**  
A. 1台から発注いただけます。なお、30台以上・100台以上でロット単価が適用されます（詳細はご相談ください）。

**Q. 在庫がない場合は？**  
A. タダカヨよりAB Circle Japanへ発注し、通常1〜2週間での納品となります。在庫状況は都度お知らせします。

**Q. キャンセル・変更は？**  
A. 出荷前であればキャンセル・変更が可能です。出荷後のキャンセルはご対応できません。
